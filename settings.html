Firebase isn't connected yet

This page edits the text and photos shown across your website, syncing instantly to every visitor on every device. To turn it on, connect a free Firebase project — it takes about 5 minutes.

  1. 1

    Go to console.firebase.google.com and create a free project.

  2. 2

    Enable Firestore Database and Authentication (Email/Password method) from the Build menu.

  3. 3

    In Authentication → Users, add yourself with an email and password — that's how you'll log in here.

  4. 4

    Register a Web app in Project Settings, copy the firebaseConfig values into the CMS_FIREBASE_CONFIG block near the bottom of every page (this file and all 6 site pages), and apply the security rules from FIREBASE-SETUP.md.

  5. 5

    Re-upload the updated files to your host, then refresh this page.

Until then, every page simply shows its normal built-in text and photos — nothing is broken.
